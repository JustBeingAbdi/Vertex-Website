Features

Add Vertex to your server to get all the following features and more!

  • Ticket System

  • Moderation and Utilty

  • Info Commands + Other Commands

Ticket Commands

Vertex Has a Fully new Ticket System with multiple Commands and Ticket Reaction panel

  • Ticket Reaction Panel. Create a Ticket By Reacting
  • v!New. Create a Ticket by doing a Command
  • Move. Move Tickets to another Catagory to make them more catagorized
  • Close. Close a Ticket with an Option to Delete and Log the Ticket
  • Rename. Rename a Ticket to make it more easier to see what the ticket is about
  • HTML Transcript. After a Ticket is deleted Vertex will generate a Transcript of the Ticket and send it to the Ticket Logs
  • Online Ticket Archive. Instead of Downloading the HTML Transcript visit the online Ticket Archive to see the Transcript instead

Keep it clean

Vertex has an extensive list of moderation commands and functions for helping you keep your server the way you want it to be.

  • Automatically grant new users a designated role
  • Kick, ban, or warn members with a simple command
  • Automatically delete messages containing your set words/phrases
  • Set up welcome/goodbye messages for when members join or leave
  • Set up a live stat tracker for your server
  • Automatically log all server activity to a designated channel
  • Bulk delete large amounts of messages

Have fun

Vertex has many miscellaneous commands that can vastly improve the user experience.

  • Vertex has many different image commands you can use!
  • Many Fun Commands you and your server can enjoy to use
  • Create a poll for users to vote on
  • View detailed information about the specified user, server, or emote
